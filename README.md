# Netflix EDA Project

This project explores Netflix’s content library using Exploratory Data Analysis (EDA).
The objective is to uncover trends in content growth, genre distribution, regional
contributions, and content freshness. The findings provide actionable business insights
that can support Netflix’s content strategy and market expansion decisions.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Movies vs TV Shows distribution
- Content growth over years
- Country-wise and genre-wise dominance
- Movie duration and TV show seasons analysis
- Content freshness trends

## Outcome
The project delivers business-driven insights that show how Netflix can improve its
content acquisition, regional expansion, and engagement strategy.
